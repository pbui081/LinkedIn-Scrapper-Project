## Python Standard Library
## Here are the libraries used in this project!
- **re** — Regular expressions
- **sqlite3** — SQLite database handling

## Third-Party Libraries (Install with pip)
- **requests** — For sending HTTP requests  
  `pip install requests`

- **beautifulsoup4** — BeautifulSoup HTML/XML parser  
  `pip install beautifulsoup4`

- **lxml** — Parser used by BeautifulSoup (optional but recommended)  
  `pip install lxml`

- **flask** — Web framework for building your server  
  `pip install flask`

## Local Imports
- `from app import app`  
  → Requires a local **app.py** or **app/** package containing `app = Flask(__name__)`
